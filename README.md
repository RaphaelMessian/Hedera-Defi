## Hedera-Defi

This repo is a list of basic DeFi smart contract on Hedera

## Setup environment

```shell
cp .env.sample .env
nano .env # edit the OPERATOR_ID and OPERATOR_KEY to match your Hedera Account and Private Key
```

## Installation

```shell
npm i
npx hardhat compile # to compile Smart Contract
```

## Run

```shell
cd scripts
node xxxxxx.js 
```
